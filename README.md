# translate-tool-chinese

## To install dependencies

```shell
npm install
```

## Examples

----------------------------------------


```shell
node chn2en.js node chn2en.js "今天星期几？今天星期一。"

# OR

node chn2en.js < demo_chn.txt
```

----------------------------------------
